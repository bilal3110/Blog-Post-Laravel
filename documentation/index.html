Installation

Download the Project

  1. Open Project in any Code Editor, and modify DB name, username, password to your environment.
  2. Install composer dependencies : composer install
  3. An application key can be generated with the command : php artisan key:generate
  4. Migrate the database : php artisan migrate
  5. Seed Demo Data into database : php artisan db:seed
  6. Run the artisan serve command : php artisan serve
  7. Proceed to : http://127.0.0.1:8000/dashboard
